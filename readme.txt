This plugin makes it possible to add a ZeroIRC Chatroom onto your Wordpress installation.

== Description ==

ZeroChat makes it possible for you to easily add a [ZeroIRC](http://zeroirc.com/ "ZeroIRC") channel (chat room) onto your Wordpress installation. ZeroChat can be added to a page simply through the options page on the Dashboard or there is also a shortcode included for use in posts and other pages. ZeroChat also includes a widget in order to place the number of current users in your chartroom on your sidebar.

== Installation ==

1. Upload all files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change the settings in WP Dashboard -> Settings -> ZeroChat
